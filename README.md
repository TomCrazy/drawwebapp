# drawwebapp
Draw Web App  
HTML5 + WebGL 2.0 + AntDesign (https://ant.design/)  
Works on the latest Chrome  
Works for Surface and Wacom tablets  
Need to start draw-server(https://github.com/vexilligera/draw-server) at localhost:3000
